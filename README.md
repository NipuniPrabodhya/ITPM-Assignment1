Installation process
1. Clone or download the project from the Git repository.
2. Open a terminal/PowerShell in the project root directory.
3. Install dependencies using npm: npm install
4. Install Playwright browsers (Chromium, Firefox, WebKit): npx playwright install
5. To run all the automated tests: npx playwright test
6. Run a specific test file: npx playwright test tests/Pos_Fun.spec.js
7. Show test results in detail: npx playwright show-report
